02. 데이터 종류 = 자료형
JS : 모든 것들이 data로 구성 및 동작
-> data 기준으로 생각하자

# String
: 따옴표@@@로 묶인 것이 하나의 문자 data
* `` (백틱) 
<보간법> 방식
: ${} 추가적인 (다른) data를 보간해서 채워넣겠다

# Number
: 정수 및 부동소수점 숫자를 나타냄
* 따옴표가 있으면 문자 ㅇㅇ "123"

# Boolean
: true/false 두 가지밖에 없는 논리 data
* 따옴표가 있으면 문자 ㅇㅇ "true"
